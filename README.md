# RayCasting

2D Ray Casting with Java. With a top down human zombie killing game built on top of it.

RayCastCore.Human controls
------------------
forward - w
strafeLeft - a
backwards - s
strafeRight - d
turnLeft - q
turnRight - e

RayCastCore.Zombie controls
------------------
forward - i
strafeLeft - j
backwards - k
strafeRight - l
turnLeft - u
turnRight - o
